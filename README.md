# Petstore142 🐾 — Testes de API com DDT

Este projeto contém uma suíte de **testes automatizados de API** para a aplicação **Petstore**, utilizando a abordagem **Data-Driven Testing (DDT)** para validar múltiplos cenários de forma eficiente.

## 📌 O que foi implementado - 
= **Testes de API REST** para os endpoints da Petstore (`/pet`, `/store`, `/user`).
- **Data-Driven Testing (DDT):** os testes são alimentados por diferentes conjuntos de dados, permitindo validar múltiplas combinações de entradas e saídas.
- **Estrutura organizada:** código-fonte em `src/` e casos de teste em `test/`.
- **Automação de validações:** status code, corpo da resposta e regras de negócio da API. 

## 🛠️ Tecnologias utilizadas
- **Java**
- **JUnit/TestNG** (framework de testes)
- **RestAssured** (para chamadas HTTP e validações)
- **Maven** (gerenciamento de dependências e execução)
